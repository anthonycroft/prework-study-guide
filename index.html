HMTL

  • We use semantic HTML elements to indicate the role of their content, as well provide structure.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The p element splits text on the page into paragraphs.
  • The section element splits the webpage into thematic sections.
  • ul and ol elements are used for adding unordered and ordered lists respectively.
  • A link element creates a connection to an external file.
  • The rel attribute specifies the relationship between the current document and linked document resource.
  • The href attribute specifies the location (URL) of the external resource using a relative path.
  • Escaping HTML characters in a string means replacing the:
    • less than symbol (<) with &lt;
    • greater than symbol (>) with &gt;
    • double quotes (") with &quot;
    • single quote (') with &#39;
    • ampersand (&) with &amp;

CSS

  • margin indicates how much space we want around the outside of an element.
  • padding indicates how much space we want around the content inside an element.
  • id and class attributes allow CSS rules to be applied to groups of elements.
  • A .class selector points to all elements that have the class attribute specified.

Git

  • git status: checks what branch we are currently in.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git rebase main: merges the MAIN branch into the feature branch.
  • git add -A: stages changes.
  • git commit -m '""': commits changes.
  • git pull origin main: is used to pull in the base branch to the feature branch.
  • git push origin feature '' is used to push up your changes to the remote repository.

JavaScript

  • Functions are reusable blocks of code that contain instructions to perform a specific task.
  • To declare a function, we use the function keyword and then give our function a unique name followed by parentheses
  • The function's executable code is written inside {} following the ().
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Math is a built-in object that has properties and methods for mathemeatical constants and functions.
